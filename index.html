Background: Extended Position-Based Dynamics (XPBD)

Representation

  • A particle $p_{i}$, has a mass $m_{i}$, a position $x_{i}$ and a velocity $v_{i}$
  • A constraint $C_j(p_{i_{1}},p_{i_{2}},...,p_{n_{j}})$ , is a function $C_{j} : R^{3} \rightarrow R $

Constraint Projection

\begin{equation} C(\textbf{x} + \Delta \textbf{x}) \approx C(\textbf{x}) + \nabla C(\textbf{x}) . \Delta \textbf{x} \succ 0 \label{eq:approx} \end{equation}

\begin{equation} \Delta \textbf{x} = \lambda \nabla C( \textbf{x}) \label{eq:lagrange} \end{equation}

Interactive Simulation

Interactive Simulation

  • design and modeling 3D objects
  • select material types, shapes, and sizes of the brush
  • 3D objects are unprovided of physical properties
  • are represented as a set of particles
  • users can touch, stretch, grasp, and move them
  • physics is assigned to the 3d object
  • objects interact with between them, with user and environment
  • based in XPBD solver
